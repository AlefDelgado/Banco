A banking system developed in Java that simulates basic financial operations using Object-Oriented Programming.

---

## 📖 Description

This project implements a banking management system that allows handling clients, accounts, and transactions.

It is designed for educational purposes to reinforce concepts such as:

* Encapsulation
* Abstraction
* Data structures
* Graphical User Interfaces (Swing)

---

## ✨ Features

* 👤 Client registration
* 💳 Bank account creation
* 💰 Deposits and withdrawals
* 🔄 Transfers between accounts
* 📊 Balance inquiry

---

## 🧠 Architecture

```bash
src/
 ├── MAIN/         # Core classes (Account, Client)
 ├── FILES/         # Bank Databas
 ├── LOGIC/         # App control
 ├── gui/           # Graphical interface
```
